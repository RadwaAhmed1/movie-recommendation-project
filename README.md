# 🎬 Movie Recommendation System
### Final Project — Data Science Course
**Dataset: MovieLens Latest Small**

---

## 📌 Project Overview

This project builds a **Movie Recommendation System** using the MovieLens dataset.  
It includes full data analysis, machine learning models, and a Streamlit web app.

**Questions answered:**
- What movie genres are most popular?
- Which movies receive the highest ratings?
- Can we recommend similar movies to users?
- What factors are related to higher movie ratings?

---

## 📁 Project Files

| File | Description |
|------|-------------|

---

## 🚀 How to Run

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Run the Jupyter Notebook
```bash
jupyter notebook Movie_Recommendation_System.ipynb
```
Run all cells top to bottom. This will:
- Clean and analyze the data
- Train all 3 models
- Save model files (`cosine_sim_df.pkl`, `movies_processed.csv`)

### 3. Launch the Streamlit Web App
```bash
streamlit run app.py
```
Then open your browser to `http://localhost:8501`

---

## 🤖 Models Used

| Model | Description |
|-------|-------------|
| **KNN** | Predicts movie ratings based on similar movie features |
| **Content-Based Filtering** | Recommends movies with similar genres using cosine similarity |
| **SVD** | Collaborative filtering — learns from user behavior patterns |
